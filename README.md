# DelayedPhoton
Plots and limits for delayed photon analysis
=============================
-----------------------------
setup
-----------------------------
```
cmsrel CMSSW_8_1_0
cd CMSSW_8_1_0/src
mkdir -p HiggsAnalysis
cd HiggsAnalysis
cmsenv
git clone git@github.com:zhangzc11/DelayedPhoton.git
cd DelayedPhoton
make
```
